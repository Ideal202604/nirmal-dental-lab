# Nirmal Dental Labs — Premium Dental Laboratory Website

A modern, premium, production-ready website for Nirmal Dental Labs — a prosthodontist-led B2B dental laboratory based in New Delhi, India. Built with React, TypeScript, Tailwind CSS, ShadCN UI and Framer Motion.

## ✨ Highlights

- Premium healthcare SaaS aesthetic — clean white + soft blue gradients, glassmorphism, soft shadows
- Full dark mode with localStorage persistence
- Smooth scroll-based animations powered by Framer Motion
- Mobile-first responsive layout
- Accessible, semantic HTML with strong heading hierarchy
- SEO-ready meta tags, Open Graph, canonical URL

## 🧩 Sections

Home · Vision · Why Us · Technology · Products · Process · Team · Booking · Contact · Footer  
Floating WhatsApp CTA · Embedded Google Map

## 🚀 Tech Stack

- React 18 · TypeScript · Vite
- Tailwind CSS · ShadCN UI
- Framer Motion (animations)
- React Hook Form + Zod (form validation)
- React Router DOM
- Lucide React (icons)

## 📦 Getting Started

```bash
bun install
bun run dev
```

Build for production:

```bash
bun run build
```
